##  Funcionamiento

  Carta estelar con capacidades de representar la posición y las estrellas en tiempo real según la ubicación del usuario dad en coordenadas y la fecha. Posición de los planetas, el sol, y la luna. información sobre dirección y ángulo respecto al horizonte para cada uno de los astros representados en el mirador, con referencia de las direcciones y habilidad de hacer zoom y mover el punto de enfoque con el ratón o el teclado.  
  Modo nocturno, ideal para uso en situación de baja iluminación y para retener la sensibilidad en la vista necesaria a la hora de utilizar un telescopio. Cambio de la interfaz a tonos de rojo para tal fin.

## Uso e instalación

1. Descargar el archivo haciendo clic en "<> Code" y luego en "download ZIP" 
2. Asegurate de tener <a href="https://www.python.org/" target="_blank">Python</a> 3 instalado en el ordenador, asi como tambien las bibliotecas<a href="https://numpy.org/" target="_blank">numpy</a> y <a href="https://www.astropy.org/" target="_blank">astropy</a>
3. Tras descomprimir el zip en una carpeta, hacer click en "Titila.py" 
4. El mapa se abrira junto con la consola.
5. Modificar la ubicación del usuario en la esquina superior izquierda para que el cielo nocturno concuerde con la aplicación. 
6. Usar el panel de control en la esquina inferior derecha para mover el mapa, o arrastrar con el ratón y hacer zoom con la rueda.
7. Las opciones para visualizar pueden cambiarse en la barra de herramientas en la parte de arriba.
8. Es recomendado cambiar el archivo "Titila.py" a uno con extension".pyw" para evitar la consola mientras se use el programa.
9. El archivo "bsc5 stars titilaformat.csv" contiene el catálogo completo (9110 estrellas). Copiar y pegar dentro de "titila.py" para alterar el número de estrellas visibles (no recomendado).
